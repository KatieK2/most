A tribute to the Mug of VI; keyboard shortcuts for the Sublime Text user.

View on Github: - includes customized keymap files.

The Mug of Sublime Text

Commands tested in Windows. For OSX, use Command in place of Ctrl, unless otherwise noted.

File Commands

Ctrl Shift t
Open most recently closed file.

Selecting

Ctrl Shift a
Select contents of tag. Again to select the tag.
Ctrl Shift m
Select contents of brackets; again to select the brackets too
Ctrl Shift Space
Select attribute value (plus quotes); again to select tag
Ctrl l
Select the current line. (Subsequent presses of L while holding Ctrl will select lines further down.)
Ctrl Shift l
Change highlighted block into a multi-selection.
Alt F3
Select all occurrences of the highlighted text; Ctrl Cmd g on OSX
Ctrl d
Add next occurence of selection as a multi-selection.
Ctrl [
When text is selected, press Ctrl Shift [ to fold that block of code out of view. Use Ctrl Shift ] to unfold.
Ctrl Alt DownArrow
Add line below as multi-selection [UpArrow for line above]
Ctrl c
Copy selected text. When nothing is selected, ST will copy/cut the current line of text. [Ctrl x to cut currently selected line.]
Control Shift Space
Select current attributes, including quotes.

Editing Text

Ctrl /
Comment the highlighted text (or current line); again to un-comment
Alt Shift w
Wrap selection with tag (Ctrl Shift w on Mac)
Ctrl Shift k
Delete current line(s)
Ctrl Shift d
Duplicate line(s)
Ctrl j
Join 2 lines of text; make current line and line below it one line
Ctrl Shift j
Join all highlighted lines into a single line
Ctrl Shift UpArrow
Swap current line (selected block) with line above [DownArrow for below]
Ctrl Backspace
Delete word in front of cursor
Ctrl Del
Delete word behind cursor
Ctrl k u
Uppercase selected text
Ctrl k u
Lowercase selected text
Ctrl k backspace
Delete from cursor to beginning of line. Cmd Delete on OSX.
Ctrl k k
Delete from cursor to end of line. Cmd k k on OSX.

Environment / View

Alt Shift 1
Switch to single column layout; Alt Cmd 1 on OSX.
Alt Shift 2
Switch to 2 column layout; Alt Cmd 2 on OSX.
Ctrl +
Increase text size
Ctrl -
Decrease text size
Ctrl k t
Fold all tag attributes.
Ctrl k 2
Fold all code deeper than 2 tags, except cursor's current location.
Ctrl k j
Unfold all folded code.

With Customized Keymap

Ctrl Shift r
Tidy Selected HTML (Edit > Line > Reindent)
Ctrl v
Newly pasted text will use existing indentation; Ctrl Shift v for standard paste.
Ctrl Shift w
Close all open files
Alt w
Toggle word wrap. Fn w on OSX.
Ctrl k i
Titlecase selected text

Wicked Cool Stuff

Emmet.io

Ctrl u
Insert dimensions into an image.
Ctrl '
Encode image as Base 64
Ctrl Alt j
Move cursor to open tag; again to move cursor to closing tag
Shift Ctrl ;
Delete parent start and end tag from cursor position.
Ctrl UpArrow/DownArrow
Increase/Decrease Value by 1; add Alt to increment by 10

See http://docs.emmet.io/cheat-sheet/ for more super powerful snippets / expansions provided by Emmet.io.

Bookmarks

Ctrl F2
Add / remove bookmark
F2
Move to next bookmark; Add Shift for previous
Alt F2
Select all bookmarks
Ctrl Shift F2
Remove all bookmarks

Sorting

F9
Sort selected lines. Ctrl F9 for case-sensitive sort. See also Edit > Permute Lines and Selection (multi-selection).

Notes

  1. "Current" means where the cursor is, or a selected block.
  2. Don't put "+" in a command unless the user has to actually hit the plus key.
  3. Use lower case letters unless upper is required.

Tips

  • Ctrl k is the "extra commands opener" which leads to a lot of awesome keyboard commands. Keep holding Ctrl / Cmd while pressing the other keys.
